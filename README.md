# element-ui-el-table-draggable

Make 'el-table' of 'element UI' sortable.

## Installation

```
$ npm i -save element-ui-el-table-draggable
```

## Quick Start

```js
import ElTableDraggable from 'element-ui-el-table-draggable';

new Vue({
  components: {
    ElTableDraggable,
  },
});
```

```html
<el-table-draggable>
  <el-table>
    ......
  </el-table>
</el-table-draggable>
```
